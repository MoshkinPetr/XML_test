В ходе выполения задания возникли проблемы с несоответствием ТЗ и предоставленного файла. Как пример, что информация об идентификаторе находится не в атрибуте, а в дочернем узле. Не было возможности уточнить, поэтому были проделанный сделащие действия. 

Для решения возникших проблем были созданы 4 списка: 

1. С названием. Для вывода в дерево.
2. С Названием выводимых узлов. Для вывода в пользовательском интерфейсе.
3. С названием дочернего узла, содержащий ID.
4. С ID.
